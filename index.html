Make.com Zero to Hero

เปลี่ยนงานที่วุ่นวาย ให้กลายเป็นระบบอัตโนมัติใน 6 ชั่วโมง

เหมาะสำหรับมือใหม่ (No-Code)

ระยะเวลาเรียน

4 - 6 ชั่วโมง

รูปแบบการเรียน

Workshop (เน้นปฏิบัติ)

ระดับความยาก

เริ่มต้น (Beginner)

Module 1

The Foundation (1 ชม.)

  • ทำความรู้จัก Automation และข้อได้เปรียบของ Make.com
  • ศัพท์พื้นฐานที่ต้องรู้: Scenarios, Triggers, Actions, Operations
  • สำรวจ Dashboard และเครื่องมือต่างๆ (Interface Walkthrough)
  • การเชื่อมต่อ Application ตัวแรกเข้ากับ Make (API Connections)
Module 2

Logic & Data Mapping (1.5 ชม.)

  • การดึงข้อมูลจากโมดูลก่อนหน้า (Data Mapping)
  • การใช้ Router เพื่อแยกสายการทำงานแบบมีเงื่อนไข (Conditional Logic)
  • การกรองข้อมูลด้วย Filters
  • การใช้ Functions พื้นฐาน: ปรับแต่งข้อความ, วันที่ และการคำนวณตัวเลข
Module 3

Advanced Control (1.5 ชม.)

  • จัดการข้อมูลชุดใหญ่ด้วย Iterators และ Aggregators
  • การตั้งเวลาทำงานอัตโนมัติ (Scheduling)
  • Error Handling: วิธีแก้ปัญหาเมื่อ Scenario ทำงานผิดพลาด
  • แนะนำโมดูล HTTP สำหรับเชื่อมต่อ API ภายนอก
Module 4

Real-world Workshop (2 ชม.)

  • สร้างเนื้อหา Blog Post จากข้อมูลจาก Google Sheet
  • เรียนรู้ไอเดียการสร้าง AI Prompt สำหรับสร้างเนื้อหา
  • เรียนรู้เรื่องการใช้ AI Model ที่เหมาะสำหรับสร้างเนื้อหา
  • Publish เนื้อหาไปยัง WordPress ด้วย Make.com

เตรียมเครื่องมือให้พร้อม แล้วมาเริ่มสร้างหุ่นยนต์ทำงานแทนคุณกันครับ
